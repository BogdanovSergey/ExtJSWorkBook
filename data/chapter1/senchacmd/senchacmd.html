Создание приложения:



Скачиваем (откуда?) SenchaCmd-7.6.0-windows-64bit.exe, устанавливаем SenchaCmd (пробовал на -6.6.0.13 или 7.6.0.87) Прописываем PATH путь к Sencha Cmd: C:/Users/%UserName%/bin/Sencha/Cmd. в переменные среды; Скачиваем файлы фреймворка ext-7.6.0.zip, разархивируем, например, сюда C:/Documents and Settings/%UserName%/sencha-sdks/ext-7.6.0; В удобном месте создаем директорию для проекта sencha_ui, заходим в неё;
sencha app init -ext='C:/Documents and Settings/%UserName%/sencha-sdks/ext-7.6.0' -classic Demo [sencha app build –dev]
запускаем sencha app watch, но при открытии приложения в app.js не находит requires: 'Ext.MessageBox' Уже можно использовать, но, удалив 'Ext.MessageBox' из app.js. sencha generate controller Control – создать файл контроллера sencha generate model --name Model --fields key:value sencha generate theme my-theme - разобрать подробнее
