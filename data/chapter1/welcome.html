Введение в Ext JS

About this Handbook

Общее:

ExtJS – начинался как библиотека красивых компонентов для разработчиков.

Фреймворк или библиотека компонентов на JavaScript, производящий Rich Internet Applications (RIA), Single Page Application (SPA); Обладает широким спектром возможностей, готовыми к использованию сразу из коробки, например: крупной библиотекой компонентов; одними из самых навороченных графиков в индустрии; быстрой стилизацией.

Целевая аудитория: корпоративные проекты, крупные сайты внутри компаний предназначенные для сотрудников.

Компоненты: В версии 7 входит более 140 готовых компонентов. Некоторые компоненты не имеют себе равных по богатству функциональности и скорости создания (Grid, Calendar). Диаграммы.

Язык: ExtJS сочетает в себе как ООП так и процедурный подходы. имеет собственную систему классов. реализует архитектуру MVC и MVVM; Кроссбраузерность, поддерживает множество мобильных браузеров. Можно использовать как в виде отдельного скрипта, так и путем сборки приложения с помощью системы сборки Sencha Cmd.

Библиотека компонентов или фреймворк? Это холиварный вопрос. Вы можете использовать Ext для отражения одного красивого компонента посреди обычного HTML и CSS, тогда это библиотека. Но я предпочитаю использовать все возможности от момента старта загрузки скриптов с приложением, до работы в области администратора.

Сложности и причины непопулярности:

ExtJs - не самый производительный из существующих фреймворков по причине избыточного DOM'а. Поэтому фреймворк не используется для обычных сайтов в Интернете. В основном для корпоративных закрытых порталов, личных кабинетов, рабочих столов, для систем с которыми постоянно работают.

Хотя компания и обещает быструю стилизацию проекта, но это только в рамках заданных тем. Чтобы добраться до html и css иногда нужно достаточно глубоко копать. Это может создать сложность в создании именно такого дизайна который нарисовал ваш дизайнер в Figma.

Высокий порог освоения: хотя всё и подчиняется правилам того же JavaScript, но серьезный проект вы не напишите без знания архитектуры и устройства компонентов.

Коммерциализация кода: начиная с версии Ext JS 5 (2014 год) начинается монетизация. Актуальные версии фреймворка можно скачать только оплатив лицензию. Несмотря на это есть возможность скачать GPL версию для личного использования.
